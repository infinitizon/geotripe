Add a badge declaring a badge attribute to the menu item.
config = { data : [ { "label" : "Mailbox", "icon" : "fa-envelope", "badge" : "16" }] }

Add a css contextual class to your badge using the badge.context attribute. In that case declare the value of the badge through the use of the badge.value attribute.

Default css declare the following badge contextes, badge-danger, badge-warning, badge-success, badge-info and badge-inverse. If none fit your needs just create your own css class.

config = { data : [ { "label" : "Mailbox", "icon" : "fa-envelope", "badge" : { "context" : "badge-success", "value" : "8"} }] }
